## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Installation

To install dependencies:

```bash
npm install
```

## Description
The project uses following stack:

- Next.js
- [Fake API](https://fakestoreapi.com)
- Headless UI
- Heroicons
